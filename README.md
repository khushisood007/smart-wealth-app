💰 Smart Wealth App — AI Investment Planner

Smart Wealth App is a financial intelligence platform that helps users analyze spending habits and generate optimized investment strategies across stocks, SIPs, gold, and fixed deposits.

✨ Why This Project?

This app helps users:

Understand their expenses clearly
Identify how much they can invest
Get structured investment suggestions
🔑 Key Highlights
📊 Expense analysis (Needs vs Wants vs Savings)
🎯 Risk-based strategy (Low / Medium / High)
📈 Multi-asset investment recommendation:
Stocks (diversified sectors)
SIP / Mutual Funds
Gold
Fixed Deposits
🔄 Real-time stock pricing integration
🧠 Portfolio visualization with charts
🧩 Core Logic
Needs → Essentials (rent, food, utilities)
Wants → Lifestyle (shopping, entertainment)
Savings → Investment potential
💡 Based on Savings, the App:
Allocates funds across asset classes
Suggests stock quantities
Calculates SIP units using NAV
Estimates gold investment in grams
⚙️ Tech Stack
Backend: Python (Flask)
Frontend: HTML, CSS
Data API: yFinance
Charts: Matplotlib
